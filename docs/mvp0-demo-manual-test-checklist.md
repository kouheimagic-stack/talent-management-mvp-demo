# MVP 0 社内デモ 手動テストチェックリスト

## ログイン

- [ ] スマホでログイン画面が崩れない
- [ ] PCでログイン画面が崩れない
- [ ] `misaki.sato@example.com / password123` でログインできる
- [ ] 一般社員でログイン後、マイページに遷移する

## ナビゲーション

- [ ] スマホでは上部の横並びメニューが表示されない
- [ ] スマホでは下部タブメニューが表示される
- [ ] 下部タブから `ホーム`、`編集`、`社員を探す`、`プレビュー` に移動できる
- [ ] 現在表示中のタブが強調される
- [ ] 一般社員に上司・人事向けメニューが表示されない

## マイページ

- [ ] マイページを見れば、このアプリの目的が分かる
- [ ] プロフィール完成度が表示される
- [ ] 公開中の項目数と非公開の項目数が表示される
- [ ] プロフィール編集への導線が分かりやすい
- [ ] 公開プロフィールへの導線が分かりやすい

## プロフィール編集

- [ ] 顔写真を選択するとプレビューされる
- [ ] 顔写真を削除できる
- [ ] 各入力欄に何を書くべきか分かる
- [ ] 公開中、非公開、非公開固定の意味が分かる
- [ ] 公開に切り替えると確認モーダルが出る
- [ ] 非公開固定項目にはトグルが出ていない
- [ ] 入力内容を変更すると「未保存の変更があります」と表示される
- [ ] 保存ボタンで保存できる
- [ ] 保存後に保存完了状態が分かる
- [ ] 保存後、リロードしても内容が残る
- [ ] スマホで保存ボタンと下部タブが重ならない

## 公開プロフィール

- [ ] 編集画面のプレビューには公開中の項目だけが表示される
- [ ] 公開プロフィール詳細には公開中の項目だけが表示される
- [ ] 空欄の公開項目は自然に非表示になる
- [ ] 非公開情報が公開プロフィールに表示されない
- [ ] 社員公開プロフィール一覧から詳細画面に移動できる
- [ ] サンプル社員が自然な内容で表示される

## 表示してはいけない情報

- [ ] 異動希望が公開プロフィールに表示されない
- [ ] 面談前メモが公開プロフィールに表示されない
- [ ] 希望キャリア非公開コメントが公開プロフィールに表示されない
- [ ] 評価、給与、面談記録、AI分析、上司コメントが公開プロフィールに表示されない

## レスポンシブ

- [ ] スマホで横スクロールが発生しない
- [ ] スマホで文字が小さすぎない
- [ ] スマホで入力欄と公開切替が押しやすい
- [ ] PC表示でナビゲーションと本文が大きく崩れない

## ビルド確認

- [ ] `npm run lint` が通る
- [ ] `npm run build` が通る
